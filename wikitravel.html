class wikitravel.travel.Travel(location, gkey, numResults=15, radius=1000, popDate='201506', factors=None, weights=None)[source]¶

Bases: object

Class for finding nearby sites of interest via google maps and wikipedia Site popularity evaluated by page views in the past (or specified month)

Create Travel object based on location, factors + weights (indev)

Parameters:
  • location (str) – string specifying location, default is geocode
  • gkey (str) – Google Maps api key
  • results (int) – max number of wiki results to query for site
  • radius (int) – search radius (meters)
  • popDate (str) – month to grab wiki page views from (YYYYMM) note month pages must exist at http://stats.grok.se/
  • factors (dict) – additional criteria INDEV
  • weights (dict) – additional criteria INDEV
